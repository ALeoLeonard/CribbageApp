# Complete UI & Settings Polish

**Goal**: Ship all partially-built UI screens and fill gaps in the settings, theme picker, statistics, and sync experience so the app feels complete.

**Success criteria**:
- Settings view has all toggles wired and functional (sound, haptics, rules)
- Theme picker shows all themes with lock/unlock state and purchase flow
- Statistics include skunk/double-skunk tracking, pegging averages, hand history
- Card sort toggle available (by rank or suit)
- Stats and purchases sync across devices via iCloud
