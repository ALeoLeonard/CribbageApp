# Missing Cribbage Rules & Play Modes

**Goal**: Add the cribbage rules and play modes that experienced players expect: skunk tracking, muggins, pass-and-play, and configurable rule variants.

**Success criteria**:
- Skunk (31+ margin) and double-skunk (61+ margin) tracked and displayed
- Muggins rule available as optional toggle
- Two humans can play on one device with a hand-over screen
- Rule variants configurable in settings

## Tickets
- [ ] ticket-skunk-tracking: Track and display skunk/double-skunk
- [ ] ticket-muggins-rule: Optional muggins with manual counting
- [ ] ticket-pass-and-play: Two-human local multiplayer
- [ ] ticket-customizable-rules: Settings toggles for rule variants
