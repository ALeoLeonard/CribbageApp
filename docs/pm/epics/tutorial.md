# Onboarding & Learning System

**Goal**: Ensure new players can learn cribbage from scratch within the app and intermediate players can improve their strategy -- the #1 missing feature vs. every top competitor.

**Success criteria**:
- First-time players complete a guided tutorial game within 3 minutes
- A static rules reference is accessible from the main menu
- Optional hint system suggests optimal plays during gameplay
- Scoring practice mode teaches hand counting interactively
